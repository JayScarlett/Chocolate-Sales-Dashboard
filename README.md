# Chocolate-Sales-Dashboard

Power BI sales analytics dashboard project focused on analyzing revenue trends, product performance, salesperson performance, and regional sales activity using interactive business intelligence reporting.

Overview

This project uses Microsoft Power BI to transform raw sales data into an interactive dashboard designed for business reporting and performance analysis.

The dashboard provides insight into:

Revenue trends over time
Top-performing salespeople
Product performance
Country-level sales activity
Shipment volume and operational metrics

The goal of this project was to demonstrate practical business intelligence and data visualization skills using a clean and easy-to-understand reporting structure.

Tools & Technologies
Microsoft Power BI
Microsoft Excel
Data Modeling

The dataset contains transactional chocolate sales data with fields including:
